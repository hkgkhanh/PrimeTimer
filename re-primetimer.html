About PrimeTimer

Introduction

PrimeTimer is a web timer for Rubik's cube and twisty puzzles speedsolvers, it provides varieties of functions:

  • Timer, inspection time + 8/12 seconds voice alert, import time by timer/typing
  • Scrambles for WCA events and 70+ puzzles & subsets, get last scramble, draw scramble
  • Mutiple sessions, set penalty of any solve, calculate mean, average...

PLease be aware that PrimeTimer is currently only available for PC users (mobile supports currently in developing). It is recommended that you should use Chrome, Edge or Cốc Cốc to browse PrimeTimer as it has only been tested on these browsers. For more information click here.

Keyboard shortcut

Key Function
Alt + " ; " Delete current solve

Alt + 0 / 1 Scramble 10x10 / 11x11
