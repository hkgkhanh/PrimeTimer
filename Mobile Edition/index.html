Introduction

PrimeTimer Mobile Edition is a web timer for Rubik's cube and twisty puzzles speedsolvers, it provides varieties of functions:

  • Timer (supports touch devices and physical keyboard), inspection time + 8/12 seconds voice alert, import time by timer/typing
  • Scrambles for WCA events and 70+ puzzles & subsets, get last scramble, draw scramble
  • Mutiple sessions, set penalty of any solve, calculate mean, average...

It is recommended that you should use Chrome, Edge or Cốc Cốc to browse PrimeTimer as it has only been tested on these browsers. For more information click here.

Touch event & Keyboard shortcut

Event Function
Touch (not move) Timer
Swipe up Import time by typing
Swipe left / right Get last / next scramble
Key Function
Alt + " ; " Delete current solve
Alt + " ' " Remove all solves in current session
Alt + [ / ] Get last / next scramble
Alt + , / . / / Set last solve's penalty to OK / +2 / DNF
Alt + " = " Add new session
Alt + " - " Delete current session
Alt + 2 ~ 9 Scramble 2x2 ~ 9x9
Alt + Q / P / C / M / S Scramble Sq1 / Pyra / Clock / Mega / Skewb
Alt + 0 / 1 Scramble 10x10 / 11x11
