About PrimeTimer

Introduction

PrimeTimer is a web timer for Rubik's cube and twisty puzzles speedsolvers, it provides varieties of functions, such as:

  • Timer
  • Inspection time
  • Voice alert of 8 seconds and 12 seconds inspection
  • Import time by timer or typing
  • Hide or show time when solving
  • Scrambles for WCA events and 70+ puzzles & subsets
  • Get last scramble
  • Draw scramble function (only available for most WCA events)
  • Mutiple sessions
  • Set penalty of any solve
  • Calculate precise mean, average...
  • Keyboard shortcuts
  • Customizable color theme

Keyboard shortcuts

Key Function
Alt + " ; " Delete current solve
Alt + " ' " Reset session
Alt + Left arrow Get last scramble
Alt + Right arrow Get next scramble
Alt + " , " Set last solve's penalty to OK
Alt + " . " Set last solve's penalty to +2
Alt + " / " Set last solve's penalty to DNF
